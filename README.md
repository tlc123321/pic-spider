# Pixiv 画师作品下载器

一个用于批量下载 Pixiv 画师作品的工具。

## 功能特点

- 支持批量下载画师的所有作品
- 自动创建以画师 ID 命名的文件夹
- 支持 HTTP/SOCKS 代理设置
- 自动跳过已下载的图片
- 显示下载进度信息

## 使用方法

1. 双击"Pixiv画师作品下载器"快捷方式运行程序
2. 输入画师ID（可从画师主页网址获取）
3. 选择是否使用代理
4. 如需代理，输入代理地址（如：http://127.0.0.1:7890）
5. 等待下载完成

## 开发相关

### 环境要求
- Node.js 14+
- npm 6+

### 安装依赖
```bash
npm install
```

### 构建
```bash
npm run build
```

### 项目结构
```
pixiv-downloader/
├── src/                # 源代码目录
├── dist/               # 构建输出目录
├── logo/              # 图标文件
├── scripts/           # 构建脚本
└── downloads/         # 下载文件保存目录
```

## 注意事项

- 请使用快捷方式运行程序
- 遵守 Pixiv 使用条款
- 下载的图片仅供个人使用
